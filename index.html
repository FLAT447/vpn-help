Настройка VPN

Как работает VPN

VPN (Virtual Private Network) — Виртуальная Приватная Сеть. По сути это отдельное зашифрованное соединение, через которое идёт весь ваш трафик и все ваши запросы.

Суть процесса:

Если вы сидите без VPN, то ваш провайдер может спокойно читать куда вы заходите и анализировать ваш трафик. Когда мы включаем VPN, мы создаём отдельный зашифрованный «туннель», который провайдер не может прочитать. Запрос расшифровывается только на сервере и идет по назначению.

Важно: остерегайтесь сайтов, не поддерживающих HTTPS! Без него данные могут быть украдены даже при использовании VPN. К счастью, таких ресурсов осталось немного.

Практическая настройка

1. Списки подписок:

Конфигурации можно взять тут: free-vpn-anti-rkn, vpn-configs-for-russia, goida-vpn-site

2. Приложения:

  • IOS - Streisand, Karing
  • Windows - v2rayN, Throne, NekoBox, Karing, singbox-launcher
  • MacOS - v2rayN, Throne, Karing, singbox-launcher
  • GNU/Linux - v2rayN, Throne, NekoBox, Karing
  • Android - NekoBoxForAndroid, NekoBoxForAndroid(Модифицированный), v2rayNG, Karing

3. Добавление подписок:

Throne: Приложение > Добавить профиль из буфера обмена > Создать новую группу подписки. Настройки > Основные настройки > Подписка > Включить интервал
NekoBox: Profiles > Add new Group > Name: Любое название > Type: Subscription > Subscription: вставляем ссылку > Ok > Переходим в группу > Profiles > Current Group > Update subscription
Karing: Добавить профиль > Импорт из буфера обмена > Примечание любое > Выбираем сервер
v2rayN: Серверы > Импорт массива URL > Группа подписки > Обновить подписку без прокси
NekoBoxForAndroid: Группы > Плюсик > Имя любое > Тип: Подписка > Ссылка для подписки > Автообновление (30)
v2rayNG: Три полоски > Группы > Плюсик > URL вставляем ссылку
Streisand: Плюсик > Добавить вручную > Тип: Subscription > URL вставляем ссылку
singbox-launcher: Скачиваем Sing-box > Скачиваем wintun.dll(только для Windows) > Скачиваем Config template > Жмём Wizard > в Subscription URL вставляем ссылку > Жмём Next > Жмём Next > Жмём Save > Жмём Start > Идём во вкладку Server > Выбираем любой сервер

4. Использование и логика запуска

1
После добавления подписки запускаем URL Тест и сортируем список по задержке.
2
Перебираем конфиги с самой маленькой задержкой. Какой будет работать — тот и используем.
3
Если не отвечает ни один конфиг — обновите подписку вручную.

5. Понятия и дополнительная настройка

DoH (DNS over HTTPS) / DoT (DNS over TLS): Технологии шифрования DNS-запросов, предотвращающие слежку провайдера.
Stack (TUN): Программная реализация сетевого интерфейса. Рекомендуется gVisor для стабильности.
MTU: Максимальный размер пакета. Значение 1500 критично для работы Hysteria2.

Рекомендуемые параметры:

  • Удаленный DNS: https://1.1.1.1/dns-query
  • Stack (TUN): gVisor
  • MTU: 1500
  • Автообновление: каждые 2 часа
